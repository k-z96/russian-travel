Проект: Путешествие по России

Третий проект "Яндекс.Практикум" о путешествиях по России. 


Используемые технологии:

    Языки HTML и CSS
    макет Figma
    Flexbox Верстка
    Grid Layout
    Медиа теги для адаптивности
    Файлы по БЭМ, nested
